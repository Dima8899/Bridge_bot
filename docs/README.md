# HotBot — Telegram → Trading Bridge

HotBot — сервис для автоматического трейдинга по сигналам из Telegram-каналов.

## Ключевая идея
Каждый пользователь:
- подключает СВОЙ Telegram-аккаунт
- выбирает ОДИН канал
- настраивает автотрейдинг
- бот читает канал от имени пользователя и исполняет сигналы

❗ Боты Telegram не используются для чтения каналов.
Используется Pyrogram (user sessions).

Логика приложения :
Пользователь заходит в бота, вводит свой номер и подключает бота к своему телеграм аккаунту. 
Вводит id канала в котором приходят сигналы, которые нужно исполнять автоматически на бинанс фьючерс.
Выбирает фильтры по которым бот будет определять сигнал ли это или нет, тикер для покупки или продажы,
плече автозакрытие и сумму сделки.
Например: 
# --------------------------------------------------
# Паттерны сигналов: можно хранить хоть 20, 
# а пользователь будет включать/выключать нужные в будущем.
# --------------------------------------------------
signal_patterns = [
    re.compile(r"#(\w+)\s+(Buy|Sell)", re.IGNORECASE),
    re.compile(r"\$(\w+)\s+(Long|Short)", re.IGNORECASE),
    re.compile(r"(\w+)\/(\w+)\s+(Buy|Sell|Long|Short)", re.IGNORECASE),
]
